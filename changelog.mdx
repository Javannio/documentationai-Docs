## Upgrade Instructions

Follow these steps to upgrade your documentation platform to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="database">
    
    Create a backup of your current `docs/` directory:
    
````bash
git checkout main
cp -r docs docs-backup-$(date +%Y%m%d)
````

  </Step>
  
  <Step title="Install Updates" icon="download">
    
    Update dependencies using your package manager:
    
    <CodeGroup tabs="npm,yarn">
```bash
npm install docs-platform@latest
```
```bash
yarn add docs-platform@latest
```
    </CodeGroup>
    
  </Step>
  
  <Step title="Rebuild and Deploy" icon="rocket">
    
    Rebuild the site and deploy:
    
````bash
npm run build
npm run deploy
````

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, test your custom components to ensure compatibility. Refer to the migration guide for v1.2.0 changes.
</Callout>
