## Document Creation and Editing Tools

Start building docs quickly with an intuitive editor supporting Markdown, code blocks, and interactive components.

<Steps>
  <Step title="Create New Doc" icon="plus">
    Navigate to your workspace and select `New Document`. Choose a template for API refs or guides.
  </Step>
  <Step title="Edit Content" icon="edit">
    Use the live preview to add headings, lists, and code. Embed images or videos directly.
  </Step>
  <Step title="Add Components" icon="components">
    Insert tabs, cards, or steps via the toolbar for richer layouts.
  </Step>
</Steps>

## Version Control for Docs

Maintain clean history without losing work. Integrate with Git for branching and merges.

<Tabs>
  <Tab title="Git Workflow" icon="git-branch">
    Push docs to your repo:
    
````bash
git add docs/features.mdx
git commit -m "Add fintech use case"
git push origin main
````

    Rollback via UI: Select commit and `Restore`.
  </Tab>
  <Tab title="Built-in History" icon="clock">
    View diffs and blame info directly in the editor. No external tools needed.
  </Tab>
</Tabs>

## Collaboration and Permissions

Invite your development team and control access granularly.

| Role | Can Edit | Can Review | Can Publish |
|------|----------|------------|-------------|
| Admin | Yes | Yes | Yes |
| Editor | Yes | Yes | No |
| Viewer | No | Yes | No |
