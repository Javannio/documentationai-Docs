---
title: Adding a New Documentation Project
description: Create and manage a new project in the Zorisord documentation portal.
---

## Quick Guide

<Steps>
  <Step title="Navigate to Projects">
    Go to the Projects section from the sidebar after logging in.
  </Step>
  <Step title="Click 'Add Project'">
    Find and click the "Add Project" button at the top right.
  </Step>
  <Step title="Fill in details">
    Provide the project name, description, and assign team members as needed.
  </Step>
  <Step title="Save and Start Documenting">
    Save your project. You can now add, edit, and organize docs for your new project.
  </Step>
</Steps>
